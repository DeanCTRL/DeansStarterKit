# Dean's Starter Kit

**Dean's Starter Kit** is a beginner-friendly Python toolkit for learning about networking, OSINT, and local network scanning.  

This project is **for learning and testing purposes only**. It demonstrates basic Python coding, network scanning, and data analysis techniques.  

## Features

- Scan your local network for devices  
- Detect device types and vendors  
- OSINT username and email lookup tools  
- Simple WiFi scanning tools  
- Real-time network dashboard (basic)  
- EXIF and file metadata parsing  

## Requirements

- Python 3.14+  
- The following Python packages:
- colorama
tqdm
requests
psutil
phonenumbers
dnspython
mac-vendor-lookup
Pillow
scapy

Install dependencies:

```bash
pip install colorama tqdm requests psutil phonenumbers dnspython mac-vendor-lookup pillow scapy


how to run?

Navigate the menu to access different tools.
